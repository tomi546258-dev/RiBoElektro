# RiBo Elektro - GitHub Pages kezdőoldal

## Fájlszerkezet

- `index.html` - a honlap fő belépőoldala
- `css/style.css` - a teljes megjelenés
- `js/site-data.js` - könnyen átírható tartalmi adatok
- `js/render.js` - a dinamikus kártyák és listák kirajzolása
- `js/main.js` - mobil menü és indulási logika
- `assets/logo.png` - a jelenlegi logó

## GitHub Pages feltöltés

1. Hozz létre egy új repository-t GitHubon.
2. Töltsd fel a teljes mappát.
3. A repository gyökerében legyen az `index.html`.
4. GitHubon menj a **Settings > Pages** menübe.
5. Source: **Deploy from a branch**.
6. Branch: **main** és a root mappa.
7. Mentsd el.

Pár perc múlva a honlap elérhető lesz a GitHub Pages URL-en.
